# toc-concepts-simulation-25
Interactive CFG Transformation and PDA Simulator with step-by-step visualization of simplification, CNF conversion, and stack-based execution. 

🚀 CFG Transformation & PDA Simulator
📌 Overview
This project is an interactive web-based application that demonstrates the relationship between Context-Free Grammars (CFG) and Pushdown Automata (PDA). It provides a step-by-step visualization of grammar transformations and stack-based computation.

🎯 Features
✅ Selection of predefined CFGs (aⁿbⁿ, Palindrome, Balanced Parentheses)

✅ Display of original CFG rules

✅ Step-by-step CFG Simplification

Removal of ε-productions
Removal of unit productions
Removal of useless symbols
✅ Conversion of CFG to Chomsky Normal Form (CNF)

✅ PDA Simulation with:

Stack operations (push/pop)
Step-by-step execution
✅ Final result: Accepted / Rejected

✅ (Optional) Automata visualization

🧠 Concepts Covered
Context-Free Grammars (CFG)
Grammar Simplification
Chomsky Normal Form (CNF)
Pushdown Automata (PDA)
Relationship between CFG and PDA
⚙️ How It Works
Select a predefined grammar
View the CFG production rules
Observe step-by-step grammar simplification
Convert the grammar into CNF
Provide an input string
Run PDA simulation
View stack operations and final result
